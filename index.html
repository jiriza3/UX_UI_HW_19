My UX Skills

Data Synthesis/Analysis

In my data work I set robust objectives in which to base my KPIs upon. Once the relevant data is gathered, it is validated and scrubbed for accuracy. The data is then analyzed and given purpose with the intention of making an impact.

UI Design

Through the use of UX research, I create UI designs within Figma and Adobe XD that best capture the expectations set by a user. Careful consideration is taken to make sure that the usability heuristics of UI design are in sync with the UX expectations of the user. Existing designs are iterative and constantly tested and researched with the users I interact with.

User Flows

I typically advocate to my users that in order to improve a process once must understand the ins and outs of the current system in play. This first starts with the user and I defining the many flows of an organization’s functions and how they intermingle with the user. From there I find that my users are able to easily identify bottlenecks in their own user flows allowing the collaborative process to begin towards building improvements.
